# Mobile App Setup Guide

## Prerequisites

Before running the mobile app, ensure the following software is installed:

* Node.js (v20 or later)
* npm
* Expo Go mobile app, Android emulator, or iOS simulator

The backend API should also be running locally.

---

# Installation

Navigate to the mobile directory.

cd mobile
npm install
npx expo start
